New Blog is live!

Jan 20, 2020 2020-01-20T19:05:00-06:00 on Blog, Update
Updated Jan 20, 2020 2020-01-20T23:08:54-06:00

Hello everyone! After a few rough months, I’m happy to be back, and, if you are one of the few (very few…) people that read my blog before, you can notice several improvements. First, now I use a dark theme cause it’s easy on the eyes, now you can explore categories, tags and archives. We have a search box that actually works, previous post and related content at the end of each post.

Also there is now a “recently updated” menu on the right sidebar along with the “trending tags”, oh! And a table of contents! on the bigger posts to help navigation.

I’m very happy with how the blog looks now on every screen-size, the old comments for each post are still there and I think everything works, but if you find something looks weird please let me know!

__PS. Stay tunned for the upcoming tutorials on jekyll, ruby on rails and more javascript coding challenges!__

Cheers!
